# О проекте:

- Название: АУРА (Автоматизированная Универсальная Риск-Аналитика)
- Версия: 1.0 pre-release
- Проект выполнен по заказу ФГБОУ ВО "Тольяттинский государственный университет" (ТГУ), Россия, обл. Самарская, г. Тольятти, 2025 г.
- Приложение предназначено для автоматизации специальной оценки профессиональных рисков на основе модифицированного метода Файна-Кинни.
- Данные об опасностях, опасных событиях и мерах профилактики для оценки взяты из Приложения 1 Приказа Минтруда РФ от 29.10.2021 N 776н "Об утверждении примерного положения о системе управления охраной труда"

# Настройка проекта

## Создате виртуальную среду Python. Рекомендуемые версии Python 3.10-3.14.
- Перед установкой проверьте, какая версия Python установлена на Вашем компьютере
```commandline
python --version
```

## Создайте директорию проекта и перейдите в неё
```commandline
mkdir path\to\your\project
cd path\to\your\project
``` 
Скопируйте файлы проекта в новую папку

## Создайте и активируйте виртуальную среду
Находясь в директории проекта, выполните команды:
```commandline
pyhton -m venv .venv
".venv\Scripts\activate.bat"
``` 
После введения команды должно появиться выражение (.venv)

## Установка зависимостей
Находясь в директории проекта с активированной виртуальной средой, выполните команду:
```commandline
pip install -r requirements.txt
```

## Запуск приложения
Для запуска приложения активируйте виртуальную среду проекта (см. пункты выше) и выполните команду:
```commandline
python -m src.script
```
